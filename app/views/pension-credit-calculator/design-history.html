  3. 14 Oct 2024

    Version 4 - Attendance Allowance devolution pilot

    {{ govukDetails({ summaryText: "Main changes", html: govukTable({ firstCellIsHeader: false, head: [ { text: "Change", classes: "govuk-!-width-one-half" }, { text: "Why we made the change" } ], rows: [ [ { html: '

  4. 14 Oct 2024

    Version 3 - Add Northern Ireland link to dropouts

    {{ govukDetails({ summaryText: "Main changes", html: govukTable({ firstCellIsHeader: false, head: [ { text: "Change", classes: "govuk-!-width-one-half" }, { text: "Why we made the change" } ], rows: [ [ { html: '

  5. 27 Aug 2024

    Version 2 - Filtering

    {{ govukDetails({ summaryText: "Main changes", html: govukTable({ firstCellIsHeader: false, head: [ { text: "Change", classes: "govuk-!-width-one-half" }, { text: "Why we made the change" } ], rows: [ [ { text: "Added GDS smart answer so that users can be filtered out earlier in the journey if shouldn't use the calculator based on their circumstances. This should make it faster for users to work out if they should submit an application." }, { html: '

    Pension Credit is now one of the main benefits which can 'passport' someone to receiving the Winter Fuel Payment.

    In October 2024, a letter was sent to over 12 million households about the WFP changes and also signpost users to the Pension Credit calculator. At the time the application Pension Credit backlog was around 10 weeks, but there was a risk it could have become significantly longer.

    There are 5 user groups that shouldn't use the calculator as it could give them the wrong outcome, so these users need to be provided with information for them to continue their journey by calling the phone lines instead.

    ' } ] ] }) }) }}
